# 需求文档

## 1. 应用概述

### 1.1 应用名称
Haosi Power B2B企业展示网站

### 1.2 应用描述
为Haosi Power（昊思能源）创建全新的B2B企业展示网站（https://haosipower.com），展示储能系统、逆变器、太阳能板三大产品线，支持多语言访问和SEO优化，配备后台内容管理系统。

## 2. 用户与使用场景

### 2.1 目标用户
- B2B采购商、经销商、代理商
- 企业客户、项目方
- 网站管理员（后台CMS用户）

### 2.2 核心使用场景
- B2B客户浏览产品信息、了解公司背景、通过WhatsApp或表单联系询价
- 管理员通过后台CMS管理网站内容、产品信息、多语言内容、SEO设置

## 4. 业务规则与逻辑

### 4.1 多语言切换规则
- 用户切换语言后，当前页面内容立即切换至对应语言版本
- 语言选择状态保存至浏览器，下次访问时自动加载上次选择的语言

### 4.2 搜索规则
- 搜索关键词匹配产品名称、产品描述、页面内容
- 搜索结果按相关度排序
- 搜索结果为空时，提示「未找到相关内容」

### 4.3 联系表单提交规则
- 表单必填项：姓名、邮箱、留言内容
- 提交成功后，数据存储至后端数据库
- 提交成功后，页面提示「提交成功，我们会尽快联系您」

### 4.4 后台管理权限规则
- 仅管理员账号可登录后台
- 管理员登录后可访问所有管理功能

### 4.5 图片资源规则
- 所有图片资源域名为cdn.staticsyy.com
- 图片上传后自动生成URL，可在前台页面引用

## 5. 异常与边界情况

| 场景 | 处理方式 |
|------|----------|
| 用户访问不存在的页面 | 显示404页面，提示「页面不存在」，提供返回首页链接 |
| 搜索关键词为空 | 提示「请输入搜索关键词」 |
| 联系表单必填项未填写 | 提示「请填写必填项」，标注未填写字段 |
| 联系表单邮箱格式错误 | 提示「请输入正确的邮箱格式」 |
| 后台登录账号或密码错误 | 提示「账号或密码错误」 |
| 后台上传图片格式不支持 | 提示「仅支持JPG、PNG、GIF格式图片」 |
| 后台上传图片失败 | 提示「上传失败，请重试」 |
| 后台编辑内容保存失败 | 提示「保存失败，请重试」 |
| 网络异常导致页面加载失败 | 提示「网络异常，请刷新页面」 |

## 6. 验收标准

1. 访客打开首页，查看Hero Banner轮播、产品分类、品牌优势、合作伙伴展示
2. 访客点击产品分类，进入产品页面，查看产品卡片（图片、名称、规格），点击询价按钮跳转至联系我们页面
3. 访客在联系我们页面填写表单（姓名、邮箱、留言），提交成功后数据存储至后端
4. 访客点击WhatsApp号码，跳转至WhatsApp聊天
5. 访客在顶部导航栏切换语言（中文/英文/西班牙语/法语/阿拉伯语），页面内容切换至对应语言
6. 访客在顶部导航栏输入搜索关键词，查看搜索结果列表
7. 管理员登录后台，编辑Banner图片和文字，保存后前台首页显示更新内容
8. 管理员在后台添加新产品（名称、描述、规格、图片），保存后前台产品页面显示新产品
